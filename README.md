# Pipecat-ts

Pipecat-ts is a TypeScript implementation of [Pipecat](https://github.com/pipecat-ai/pipecat). It is a framework for building real-time voice and multimodal conversational AI agents.

## Features

- Real-time voice and multimodal conversational AI agents
- Easy to use and extend
- Based on the Pipecat architecture
- Designed to be easy to use and extend
